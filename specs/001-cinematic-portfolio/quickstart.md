# Quickstart Guide

## Prerequisites
- Node.js 18+ (Recommended: 20 LTS)
- pnpm (Preferred) or npm

## Installation

```bash
# Install dependencies
pnpm install
# or
npm install
```

## Development

Start the development server with TurboPack (default in Next 15+):

```bash
pnpm dev
# or
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Testing

Run the Playwright end-to-end tests:

```bash
# Run all tests
npx playwright test

# Run UI mode
npx playwright test --ui
```
